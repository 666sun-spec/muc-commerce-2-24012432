## 用自己的话回答

- 什么是特征，什么是标签：特征就是我们能拿到的、用来做判断的各类原始数据，拿用户流失项目举例，像用户登录次数、消费金额、最近一次活跃间隔这些都属于特征，是模型做判断的依据；标签是每条数据对应的真实结果，这里只有两种：流失、未流失，相当于给模型看的标准答案。
- 为什么要保留测试集：训练集是专门给模型用来学习规律的，如果只用训练集评估效果，模型会记住训练数据里的细节，分数会虚高。测试集全程不参与模型训练，是完全陌生的数据，用它来检验模型，才能看出模型面对新用户时真实的预测能力，判断模型会不会过拟合。
- 为什么83%准确率仍可能没有用：大多用户都是未流失的，数据分布很不均衡。比如100个用户里只有17个流失，模型只要全部猜“未流失”，就能拿到83%准确率，但它完全识别不出真正流失的用户。我们做这个项目的核心目标是找出流失人群，光看准确率没法衡量找流失用户的能力，所以这个高准确率没有实际业务价值。
